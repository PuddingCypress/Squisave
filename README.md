# 🐧 SquiSave 自动备份工具
一个简洁好用的可视化文件夹自动备份小工具 ✨

## 🚀 功能介绍
- 📂 手动选择文件夹一键备份
- ⏰ 支持每日 / 每周 / 每月定时自动备份
- 🌏 中文 + English 双语言切换
- 🎨 清爽好看的图形界面
- 📝 实时日志查看与记录
- ⌨️ 快捷键操作（Ctrl+B 快速备份）

## 🛠️ 运行环境
- Python 3.x
- PyQt5

## 📌 项目特点
- 操作简单，小白也能轻松使用
- 自动生成 ZIP 压缩包
- 配置自动保存，无需重复设置
- 支持 Windows 系统定时任务
